Rating Pathway

OBS → C3 Progression

OBS
Observer
STARTING POINT
All VATSIM members start as Observer. Complete VATSIM New Member Orientation.
S1
Developing Controller
VATSIM S1
Delivery (DEL) and Ground (GND) positions. Radio telephony, ground movement, clearances.
Positions: DEL, GND at authorised airports
S2
Aerodrome Controller
VATSIM S2
Tower control added. Circuit management, runway operations, wake turbulence separation.
Positions: TWR + all S1
S3
Terminal Controller
VATSIM S3
Approach and Departure. Radar vectoring, holding, arrival sequencing.
Positions: APP, DEP + all S2
C1
Enroute Controller
VATSIM C1
Full enroute control. Centre positions, oceanic with endorsement. All ATC facilities.
Positions: CTR + all S3
C3
Senior Controller
VATSIM C3 — AWARDED
Awarded by VATPAC for seniority and distinguished service. Equivalent to C1 functionally.
Study Material

ATC Curriculum

ATC-01Foundation — Towards S1
+
Module 1 — ATC Fundamentals
Role of ATC — separation, flow management, information
Australian airspace classification — Class A, C, D, E, G
VHF communication — frequencies, transmission technique
ATIS construction and broadcast
Module 2 — Ground Operations (S1)
Clearance Delivery — IFR clearance format (CRAFT)
SID assignment and initial altitudes
Squawk code assignment
Ground movement control — taxi routes, hold short instructions
ATC-02Aerodrome Control — Towards S2
+
Module 1 — Tower Operations
Runway management — landing clearances, line-up and wait
VFR circuit management — sequencing, extend downwind
Wake turbulence separation — categories and minima
Low visibility operations — Cat II/III procedures
ATC-03Terminal Control — Towards S3
+
Module 1 — Radar Fundamentals
Radar identification — SSR, surveillance radar
Radar separation standards — longitudinal and lateral
Vectoring for approach — intercept angles, speed control
Module 2 — Approach Sequencing
STAR management — speed restrictions, altitude crossing
Holding — assigning, managing, dissolving
Parallel runway operations — YSSY 34L/R
ATC-04Enroute Control — Towards C1
+
Module 1 — Enroute Operations
ATS route structure in Australian airspace
Conflict detection — lateral, vertical, longitudinal
RVSM airspace — 1,000 ft separation above FL290
Module 2 — Oceanic Control
AUSOTS track allocation and oceanic clearances
CPDLC and ADS-C procedures
HF voice position reports and oceanic re-clearances
