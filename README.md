## 🚀 Deployment

This site is deployed automatically via **GitHub Pages**.

| Setting | Value |
|---|---|
| Source | `main` branch |
| Folder | `/ (root)` |
| Live URL | `https://qantas-virtual-airline.github.io/Qantas-Virtual-Website` |

Any commit pushed to `main` will automatically update the live site within 1–2 minutes.

---

## ✏️ Making Changes

To update the website:

1. Edit `index.html` directly on GitHub (click the file → pencil icon)
2. Or clone the repo, make changes locally, and push to `main`
3. GitHub Pages will redeploy automatically

**Key things to update:**
- Discord invite link — search for `Join Discord — Apply Now` in `index.html` and replace `href="#"` with your actual invite link
- Pilot count — search for `5 Active Pilots` and update as the VA grows
- Route network — find the route tables and add/remove routes as needed
- Document links — all doc cards link to the Terms-and-condition repo; update if you move docs
