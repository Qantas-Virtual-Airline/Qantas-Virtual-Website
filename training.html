Curriculum Overview

Training Phases

PHASE 0New Member Orientation — P0
+
ℹ
Complete the VATSIM New Member Orientation. Set up SimBrief. Install your VATSIM pilot client.
VATSIM Setup
Create VATSIM account and obtain your CID
Complete the VATSIM New Member Orientation Course
Install and configure a VATSIM pilot client (vPilot/xPilot/Swift)
Connect to VATSIM for the first time with a mentor
QFV Onboarding
Set up SimBrief account and generate a practice OFP
Install Flight Logger access via QFV Discord
Read the QFV Pilot Handbook (POL-001) and Route Network Policy (POL-002)
Complete QFV Orientation module on Moodle
PHASE 1Private Pilot Licence — P1
+
ℹ
Target aircraft: B737-800. VFR fundamentals, basic airmanship, first solo VATSIM flights.
Theory
Theory of flight — lift, weight, drag, thrust
Meteorology — reading METAR/TAF, VFR minima
Navigation — aeronautical charts, dead reckoning
Radio communications — phraseology, readbacks
Practical (×6 supervised)
Flight 1: Straight & level, turns, climbs, descents
Flight 2: Circuit training — takeoff, pattern, landing
Flight 3: Cross-country VFR navigation
Flight 4: Emergency procedures — engine failure simulation
Flight 5: Full cross-country with ATC
Flight 6: P1 skills test preparation
PHASE 2Instrument Rating — P2
+
ℹ
Target aircraft: B737-800, A321XLR. Full IFR operations, SID/STAR, instrument approaches.
IFR Theory
Instrument scanning and cross-checking technique
SID/STAR procedures — reading and flying
ILS, RNAV/GNSS approaches — full procedure
Holding patterns — entry, timing, corrections
SimBrief IFR planning — fuel, alternates, routing
Practical (×6)
Flights 1-3: IFR departures, en-route, holds
Flights 4-5: Full ILS and RNAV approaches
Flight 6: IR skills test preparation
PHASE 3Commercial Multi-Engine — P3
+
ℹ
Target aircraft: A330-200/300. Multi-engine ops, ETOPS, oceanic. Unlocks international routes.
Theory
Multi-engine aerodynamics — yaw, Vmc, critical engine
Airbus fly-by-wire — normal/alternate/direct law
ETOPS-180 planning requirements
Oceanic procedures — CPDLC, position reports, HF
Practical (×5)
Flights 1-3: A330 checkout, engine-out procedures
Flight 4: Full oceanic crossing with CPDLC
Flight 5: CMEL skills test preparation
PHASE 4Airline Transport Pilot — P4
+
ℹ
Target aircraft: B787-9. Advanced airmanship, ultra-long-haul, complex systems.
Theory
Threat & Error Management (TEM)
Crew Resource Management (CRM)
787-9 systems — no-bleed architecture, EICAS
Ultra-long-haul fuel planning — PER–LHR methodology
AUSOTS/PACOTS/NAT track systems
Practical (×5)
Flight 1: B787-9 checkout
Flights 2-3: Long-haul oceanic crossings
Flight 4: Diversion and fuel emergency scenario
Flight 5: ATPL skills test preparation
PHASE 5A380 Endorsement + FI/FE
+
ℹ
Prerequisite: P4 (ATPL). Staff review required before endorsement.
A380 Endorsement
Systems differences from A330 — 4 engines, 8 hydraulics, trim tank
SUPER wake turbulence — filing A388/H, separation requirements
Code F airports only — infrastructure check
Supervised check flight: SYD–LHR with QFV examiner
FI (P5) / FE (P6) Pathway
Instructor techniques — brief, debrief, mentor
QFV Examiner Standards Manual
Complete P5/P6 through approved VATSIM ATO
